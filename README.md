# NotifyAppWithSignalR

You need to add a appsettings.json file to the root folder of project. Then add db connections tring as below.
{
"ConnectionStrings": {
"DefaultConnection": ""
},
"Logging": {
"LogLevel": {
"Default": "Information",
"Microsoft.AspNetCore": "Warning"
}
},
"AllowedHosts": "\*"
}
